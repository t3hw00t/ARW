Guided setup

Bring the Control Room online in three steps

Start the service, secure it with your admin token, then open the workspace that fits your project.

  1. 1

    Start the local service

    Launch arw-server on this machine. We monitor health automatically and announce changes.

    Need more context? The Debug UI mirrors the browser panels and the log shortcut opens the current stdout/stderr file.

    Prefer scripts? Restart headless with bash scripts/start.sh --service-only --wait-health --admin-token "$ARW_ADMIN_TOKEN".

  2. 2

    Secure access with an admin token

    Paste the token once to unlock Hub, Chat, Training, and other admin tooling. We keep it local and never echo it back.

    Store the token once, then use “Test” to confirm access before restarting. “Copy restart” builds a shell-ready snippet that exports the token for CLI scripts.
